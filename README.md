# Castia Client

**Castia Client** is a client-side utility client developed specifically for use on **CastiaMC**.  
Its goal is to provide **server-approved quality-of-life and visual enhancements** while strictly avoiding any unfair gameplay advantages.

Castia Client focuses on features that improve visibility, usability, and the overall player experience **without modifying combat, movement, or network behavior**.
